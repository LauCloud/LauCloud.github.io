学生管理系统(控制台版本)

发表于 2017-05-28 | 分类于 编程 | | 阅读次数

这个系统提供的主要功能是查看所有学生信息、增加学生信息、删除学生信息和修改学生信息，所有学生的信息都是保存在一个ArrayList集合之中，然后操作这个集合来进行各项功能，完成了主要了功能之后，就可以进一步地去完善了，比如加一个登录验证，还有一些细节也需要注意，比如设置学号不能为空和不能重复。

1.新建一个Student学生类

这个主要是对学生信息的管理，当然就少不了学生类了，主要的属性为学号(id)、姓名(name)、年龄(age)和居住地(address),定义类时当然也少不了构造器函数，以及对于属性的set()和get()方法,将完整的Student类的定义放在下面：

public class Student {
    private String id;
    private String name;
    private String age;
    private String address;

    public Student() {
        super();
    }

    public Student(String id, String name, String age, String address) {
        super();
        this.id = id;
        this.name = name;
        this.age = age;
        this.address = address;
    }

    public String getId() {
        return id;
    }

    public void setId(String id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public String getAge() {
        return age;
    }

    public void setAge(String age) {
        this.age = age;
    }

    public String getAddress() {
        return address;
    }

    public void setAddress(String address) {
        this.address = address;
    }
}

2.主界面的展示

因为是控制台版本的系统，所以展示也相对简陋，主要是将几个选项打印到控制台，然后有一个接受输入的光标，因为只有当用户确认想要退出的时候才会退出系统，因此需要将这个选择做成一个循环的，因为毕竟是要和用户有交互的，不能只有一次输入和输出，

while (true) {
    System.out.println("-----欢迎进入学生管理系统-----");
    System.out.println("1.显示所有学生信息");
    System.out.println("2.增加学生信息");
    System.out.println("3.删除学生信息");
    System.out.println("4.修改学生信息");
    System.out.println("5.退出");
    System.out.print("请输入您的选择:");
    Scanner sc = new Scanner(System.in);
    String choice = sc.nextLine();
    switch (choice) {
    case "1":
        showAllStudent(arr);
        break;
    case "2":
        addStudent(arr);
        break;
    case "3":
        deleteStudent(arr);
        break;
    case "4":
        updateStudent(arr);
        break;
    case "5":
    default:
        System.out.println("谢谢您的使用！");
        System.exit(0);
        break;
    }
}

3.查看所有学生信息

上面的switch语句中也可以看到，每当用户输入一个指令，系统就会执行相应的功能，最开始的时候也说了，这个系统现在所有的学生信息都是放在ArrayList中，因此应该在最开始定义一个集合。

ArrayList<Student> arr = new ArrayList<Student>();

这样所有的查看、增加、删除以及修改操作都会对这个集合产生作用，现在就说明一下查看集合中的所有学生信息，最重要的是做一个判断，有信息的话就循环遍历输出，没有的话就要给出提示，当然输出的时候也要注意格式，最好能用制表符分割(“\t”)一下。

    public static void showAllStudent(ArrayList<Student> arr) {
    if (arr.size() == 0) {
        System.out.println("抱歉！还没有录入学生信息，请重新选择操作！");
        return;
    }
    System.out.println("学号\t姓名\t年龄\t居住地");
    for (int i = 0; i < arr.size(); i++) {
        Student s = arr.get(i);
        System.out.println(s.getId() + "\t" + s.getName() + "\t"
                + s.getAge() + "\t" + s.getAddress());
    }
}

4.增加学生信息

提示用户要输入相应的属性信息：学号、姓名、年龄和居住地，然后用用户输入的这些信息去新建一个学生对象，之后就是保存到集合当中。其中需要注意的两个点是：学号不能为空和不能重复，判断重复的话就需要先遍历一下集合中已有的学生对象，当然出现这两种情况的时候就要及时给出提示，并且让循环继续。

public static void addStudent(ArrayList<Student> arr) {
    Scanner sc = new Scanner(System.in);
    String id;
    while (true) {
        System.out.println("请输入学生学号:");
        id = sc.nextLine();
        if(id.isEmpty()){
            System.out.println("学生学号不能为空！请重新输入！");
        }else{
        boolean flag = false;
        for (int i = 0; i < arr.size(); i++) {
            Student s = arr.get(i);
            if (s.getId().equals(id)) {
                flag = true;
                break;
            }
        }
        if (flag) {
            System.out.println("该学号" + id + "已存在对应的学生信息，请重新输入!");
        } else {
            break;
        }
        }
    }

    System.out.println("请输入学生姓名:");
    String name = sc.nextLine();
    System.out.println("请输入学生年龄:");
    String age = sc.nextLine();
    System.out.println("请输入学生居住地:");
    String address = sc.nextLine();

    Student s = new Student(id, name, age, address);
    arr.add(s);
    System.out.println("学生信息添加成功！");
}

5.删除学生信息

删除学生信息的话是根据用户输入的学号去删除，因此需要先判断一下集合中是否存在该学号对应的学生信息，不存在的话就要给出相应的提示，有的话就应该执行删除操作。

public static void deleteStudent(ArrayList<Student> arr) {
    Scanner sc = new Scanner(System.in);
    System.out.println("请输入要删除学生的学号:");
    String id = sc.nextLine();
    int index = -1;
    for (int i = 0; i < arr.size(); i++) {
        Student s = arr.get(i);
        if (s.getId().equals(id)) {
            index = i;
            break;
        }
    }
    if (index == -1) {
        System.out.println("该学号" + id + "对应的学生信息不存在，请选择其它操作！");
    } else {
        arr.remove(index);
        System.out.println("删除学生信息成功!");
    }
}

6.修改学生信息

修改学生信息也是根据用户输入的学号，如果该学号确实存在，那就要提示用户输入新的属性信息进行修改，不存在的话，就应该提示没有对应的用户信息。

public static void updateStudent(ArrayList<Student> arr) {
    Scanner sc = new Scanner(System.in);
    System.out.println("请输入要修改学生的学号:");
    String id = sc.nextLine();
    int index = -1;
    for (int i = 0; i < arr.size(); i++) {
        Student s = arr.get(i);
        if (s.getId().equals(id)) {
            index = i;
            break;
        }
    }
    if (index == -1) {
        System.out.println("该学号" + id + "对应的学生信息不存在，请重新选择操作！");
    } else {
        System.out.println("请输入新姓名:");
        String name = sc.nextLine();
        System.out.println("请输入新年龄:");
        String age = sc.nextLine();
        System.out.println("请输入新居住地:");
        String address = sc.nextLine();

        Student s = new Student(id, name, age, address);
        arr.set(index, s);
        System.out.println("修改学生信息成功！");
    }
}

7.增加登录检验功能

上面的就是对四个主要功能的描述，完成了上面的功能之后，我还在系统的最前面添加了这个登录功能，用户有三次机会，没输错一次机会便会减1，当然，如果输入正确，就会进入到后面的管理功能，也直接把代码放到这里吧。

for (int i = 0; i < 3; i++) {
    System.out.println("---欢迎登陆学生管理系统---");
    Scanner sc = new Scanner(System.in);
    System.out.print("用户名:");
    String user = sc.nextLine();
    System.out.print("密码:");
    String password = sc.nextLine();
    if (!(user.isEmpty()) && !(password.isEmpty())) {
        if (user.equals("root") && password.equals("123")) {
            System.out.println("恭喜您！登陆成功!");
            break;
        } else {
            if (2 - i == 0) {
                System.out.println("用户名或密码错误！请重新输入！");
                System.out.println("该账号已被锁定！请与管理员联系！");
                System.exit(0);
            } else {
                System.out.println("用户名或密码错误！请重新输入！");
                System.out.println("您还有" + (2 - i) + "次机会！");
            }
        }
    } else {
        if (2 - i == 0) {
            System.out.println("用户名或密码不能为空！请重新输入！");
            System.out.println("该账号已被锁定！请与管理员联系！");
            System.exit(0);
        } else {
            System.out.println("用户名或密码不能为空！请重新输入！");
            System.out.println("您还有" + (2 - i) + "次机会！");
        }
    }
}
